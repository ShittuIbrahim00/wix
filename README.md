# GridSpace UI

A clean, responsive landing page built with **React**, **TailwindCSS**, and **Framer Motion**, inspired by a Figma concept.

---

## Stack

- ⚛️ React (JSX)
- 💨 TailwindCSS
- 🎞️ Framer Motion
- 📦 Lucide Icons
- 🔠 Google Fonts

---

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
